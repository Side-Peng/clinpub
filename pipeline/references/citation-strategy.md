# 引用策略

## 第一章：引用总量约束

- 总引用量 **30-55 篇范围制**，此为硬约束（不可突破）
- 不设固定目标值（不做"must be 50"），讨论时根据实际情况确定
- 此范围适用于所有研究类型的 IMRAD 手稿
- 讨论时与用户确认总量偏好，在 30-55 范围内确定具体目标值

## 第二章：各段引用配比

引用配比作为弹性建议（非硬约束），各段可 ±20% 浮动。

| 段落 | 建议引用数 | 弹性范围（±20%） | 说明 |
|------|-----------|-----------------|------|
| Introduction | 10-15 | 8-18 | 背景(3-5) + 已知证据(3-5) + gap(2-3) + 目的(1) |
| Methods | 3-5 | 2-6 | 方法学来源、已发表协议、诊断标准引用 |
| Results | 0-3 | 0-4 | 仅当结果需要与已有文献对比时引用 |
| Discussion | 15-25 | 12-30 | 对比(8-12) + 机制(3-5) + 临床意义(2-3) + 局限(1-2) |

**总量验证**：各段实际引用数之和必须在 30-55 范围内。如果弹性调整导致超限，优先缩减 Discussion 段的引用量。

## 第三章：文献年限策略

- **默认**：近 5 年文献
- **例外**：landmark 经典文献不受年限限制，即使超过 5 年也可引用

**Landmark 判定标准**：
- 该领域首次报告该发现的原创论文
- 被引 >1000 次的方法学论文
- 临床指南引用的核心文献
- 系统综述/Meta 分析中的关键文献

讨论时与用户确认是否有 landmark 例外需要添加。用户可指定特定文献作为年限例外。

## 第四章：影响因子偏好

- IF 偏好无严格默认值，每次写作前与用户讨论确定
- 讨论参考：目标期刊的 IF 范围、领域内高 IF 期刊列表
- IF 偏好影响 reference-agent 搜索时的文献筛选，但不作为硬性排除条件（低 IF 但高相关性的文献仍可收录）
- 当 `min_if` 设置后，搜索结果中 IF >= 此值的文献优先保留
- 无 IF 信息的文献标记为 "IF unavailable" 供用户自行判断

## 第五章：引用格式

- **格式**：Vancouver 编号制，正文 `[1]` `[2]`，末尾统一 References 区
- 具体格式规范见 `pipeline/references/reference-library.md` §2
- 所有引用必须有 DOI
- 全局统一编号，引用于正文章首次出现顺序排列

## 第六章：讨论流程

引用策略讨论发生在 writing workflow 的 `discuss_citation_strategy` 步骤。

**讨论顺序**：
1. **各段引用数量**：以默认配比为基础，确认是否需要按实际情况调整
2. **文献年限范围**：确认近 5 年默认值，讨论 landmark 例外
3. **IF 偏好**：讨论目标期刊和 IF 下限偏好

**讨论结果写入**：`project_config.yml` 的 `citation_strategy` 段。

后续 `reference_pre_search` 和 `sequential_section_writing` 步骤读取此配置决定搜索参数和引用量。

## 第七章：与已有规范的层级关系

- `citation-strategy.md` 定义**"量"**的层面（多少篇、多新、多高 IF）
- `reference-library.md` 定义**"质"**的层面（JSON 格式、编号、去重规则、Vancouver 格式规范）
- 两者互补，共同构成完整的引用管理体系
- `discuss_citation_strategy` 步骤确定数量/年限/IF 参数
- `reference_pre_search` 步骤使用这些参数执行实际搜索和筛选
- `sequential_section_writing` 步骤在撰写时按配比控制各段引用量

## 附录：常见场景速查

| 场景 | 建议操作 |
|------|---------|
| 用户说"随便，你定" | 使用默认配比 + 近 5 年 + 无 IF 下限 |
| 目标期刊 IF 很高（>5） | 建议提高 IF 偏好至 3+，重点引用高 IF 期刊文献 |
| 领域更新很快（如肿瘤免疫） | 建议缩窄年限至 3 年，增加 Discussion 中近 1 年文献比例 |
| 领域经典文献多（如诊断标准） | 添加 landmark 例外列表，保留经典引用 |
| 用户说"多引用一些" | 在 30-55 范围内上调总量，优先扩展 Intro 和 Discussion 段 |
| 用户说"少引用一些" | 在 30-55 范围内下调总量，优先缩减 Discussion 补充对比部分 |
